#  Парсер из HTML table в txt
## Постановка задачи
Сделать парсер, который будет на вход получать table с tr и td (файл в htmlформате). Нужна поддержка атрибутов clospan и rowspan. На выходе нужно получить табличку, нарисованную текствовыми символами.
# Реализация
## Архитектура
Для выполнения задания была выбрана технология Data Driven Testing  – это подход к тестированию, при котором тестовые данные хранятся отдельно от скриптов, в документе html.
Было написано несколько тестов:


* 1. таблица
* 2-*. таблица с colspan
* 3-*. таблица с rowspan и colspan - на этом остановился.

## Запуск приложения из командной строки
проект выполнен с использованием maven 
```
$ mvn clean install
$ mvn test
```
